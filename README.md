# SnippetMate

**Paste it. Tag it. Find it. Export it.**

SnippetMate is a Single-Page Web Application (SPA) for capturing and retrieving short text or code snippets from the web, as well as AI prompts and AI responses. Tag your snippets, then search by title, tag, or full-text and export them as clean Markdown files.

Your "SnippetMate" lives in your browser while you're surfing news, X.com, or coding sites.

## Tech Stack

- **Frontend:** Vue 3 + Vite + Bootstrap 5 (Bootswatch United theme)
- **Backend:** Node.js + Express
- **Database:** MySQL 8 (FULLTEXT indexing, many-to-many snippet↔tag schema)

## Dynamic Aspects

1. CRUD operations for snippets (Create, Read, Update, Delete)
2. Server-side URL title fetching
3. Markdown export

No external APIs are used.
